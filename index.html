Generate Salesforce Fields In Bulk

Generate Salesforce fields in bulk—upload a CSV of field definitions and instantly create CustomField XML .field-meta.xml files organized by object, ready for deployment.

Important: Leave unused cells blank To avoid errors and keep your CSV clean, leave cells empty for columns that don’t apply to the selected field Type. The generator only uses the columns required by each data type and ignores the rest.
After generating the .field-meta.xml files, copy/paste them into your Salesforce project under: force-app/main/default/objects//fields/
Then deploy to your org using Salesforce CLI: sf project deploy start --source-dir force-app --target-org OR VS Code (easy way): Right‑click the folder/file (e.g., force-app or the object folder) → Deploy Source to Org
Please choose a CSV file first.
Output base dir (in ZIP)

This tool is built to generate bulk Salesforce field metadata from a single CSV. Each row becomes a CustomField XML file, automatically placed in an object-based folder structure for easy deployment.
